About the Project

Agentic AI, built for the field

The AI-Powered Smart Farming Advisor is an agentic AI platform built on IBM watsonx, designed to give every farmer access to expert-level guidance. Instead of a single chatbot, the system coordinates multiple specialized AI agents — each trained on a specific domain of agriculture — to deliver grounded, farmer-friendly answers.

Using Retrieval-Augmented Generation (RAG), the assistant draws on agricultural knowledge bases, live weather data and market feeds to keep recommendations current, while IBM watsonx Orchestrate manages how agents collaborate behind the scenes.

What makes it different

  • AI-powered agriculture — decisions grounded in real agronomic data, not guesswork.
  • IBM watsonx foundation — enterprise-grade orchestration and language models.
  • Multiple AI agents — each specialized in one part of the farming journey.
  • Retrieval-Augmented Generation — answers backed by trusted agricultural sources.
  • Farmer-friendly guidance — plain language, in the farmer's own tongue.
⚠️

Recommendations regarding fertilizers, pesticides, diseases, weather, and market prices should always be verified using official agricultural authorities, local agricultural experts, and product labels before implementation.
